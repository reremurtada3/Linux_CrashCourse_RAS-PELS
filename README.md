## USB Live Demo Instructions:
1. Connect to `AirRowdy`

2. Run this command(creates the mirror list for apps-repos)
```bash
sudo reflector --latest 10 --sort rate --save /etc/pacman.d/mirrorlist
```
3. Run this command:
```bash
sudo pacman -Syy
```
